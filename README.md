### Display image from d435i by using ros
```  
cd src/rs_d435i/
source create_catkin_ws.sh
cd ../..
. /devel/setup.bash
roslaunch realsense2_camera rs_rgbd.launch
```

open new terminal

```
. /devel/setup.bash 
. ../catkin_workspace/install/setup.bash --extend
rosrun get_rs_image Get_Image.py
```
